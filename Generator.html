How It Works

The circle is divided into 4 equal sections, each representing a range of 72 numbers (0-71, 72-143, 144-215, 216-287).

Pink String: Current string from current position to next position

Blue String: Previous string from previous position to current position

Yellow Dot: Both strings intersect at the current position

Center shows: Position within the current section (0-71)
